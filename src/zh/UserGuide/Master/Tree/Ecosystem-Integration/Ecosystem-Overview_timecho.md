# 概览

IoTDB 生态集成打通时序数据全链路：通过数据采集实现设备秒级接入，经数据集成构建跨云管道，依托编程框架快速开发业务逻辑，结合计算引擎完成分布式处理，通过可视化与 SQL 开发实现分析策略，最终对接物联网平台完成边云协同，构建从物理世界到数字决策的完整智能闭环。

![](/img/eco-overview.png)
